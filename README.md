Xây dựng website

Đối với người dùng: Cung cấp chi tiết các địa điểm du lịch, người dùng có thể xem và bình luận.

Đối với quản trị: Quản lý hiệu quả danh sách điểm đến, thêm nội dung upload hình ảnh điểm du lịch. 

Về kỹ thuật: Áp dụng mô hình MVC (Model-View-Controller) để xây dựng ứng dụng web có cấu trúc rõ ràng, dễ bảo trì và mở rộng.

Công nghệ sử dụng Ngôn ngữ: C# (C-Sharp) Framework: ASP.NET MVC ORM: Entity Framework Cơ sở dữ liệu: SQL Server Giao diện: HTML, CSS, JavaScript, Bootstrap. Thư viện soạn thảo văn bản CKEditor

Các tính năng chính:

Phân hệ người dùng: trang chủ, xem chi tiết địa điểm du lịch, tương tác bình luận, form liên hệ.

Phân hệ quản trị viên: đăng nhập, quản lý danh sách điểm đến, thêm sửa nội dung và upload ảnh địa điểm du lịch, quản lý liên hệ bình luận
